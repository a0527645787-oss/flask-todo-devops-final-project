Project overview

About this demo app

This is a friendly seat-booking demo inspired by the energy of a global football tournament. It lets visitors explore a sample match schedule, compare seating options, and walk through a simple booking journey.

Demo disclaimer

This website is a demo project created for learning and presentation purposes. It is not an official ticketing website, does not sell real tickets, and has no affiliation with FIFA, any tournament organizer, any national team, or any real ticketing system. Bookings shown in the app are for demonstration purposes only.

What regular users can do

  • Browse World Cup inspired matches.
  • Choose a match and seat type.
  • Create a booking and receive a booking code.
  • Manage a booking with booking code and email.
  • Cancel an existing booking.

What admin can do

Admin can view all bookings and basic match statistics, including booked, cancelled, and available seats.

Technical flow

Browser/User -> Flask -> SQLAlchemy -> MySQL

DevOps flow

Developer -> GitHub -> GitHub Actions -> Pytest -> Docker Build -> Docker Hub -> Deployment
